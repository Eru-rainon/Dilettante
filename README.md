# Dilettante

A personal game catalogue app built with WPF and C#.

## What it does
- Search Steam for any game and save it to your local library
- Track your status — Playing, Completed, Wishlisted, Paused, or Abandoned
- Manually track achievements with a checklist
- See completion percentage and status at a glance in the library view
- Log how you played — Owned Copy, Subscription, or Seafared 🏴‍☠️ ;)

## Built with
- C# / WPF (.NET 8)
- Entity Framework Core + SQLite
- Steam Web API
- Newtonsoft.Json
